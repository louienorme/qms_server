# qms_server

### How to run

#### Dev Environment 
1. Run `yarn install` or `npm install`
2. Modify .env to match your DB credentials
3. Run `yarn start:dev` 

### Production Environment
1. Run `yarn install` or `npm install`
2. Modify .env to match your DB credentials
3. Run `yarn build`
4. Run `yarn start:prod` 
5. Run `yarn seed`
